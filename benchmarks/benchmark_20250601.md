# Test Environment

* Date: 2025.06.01
* OS:  Win11 Pro 24H2 (26100.4202)
* CPU: [i9-14900K](https://www.intel.com/content/www/us/en/products/sku/236773/intel-core-i9-processor-14900k-36m-cache-up-to-6-00-ghz/specifications.html)
* Compiler: gcc version 15.1.0 (Rev5, Built by MSYS2 project)
* Msys2: msys2-x86_64-20250221
* Build:
  * [Build and use pe's dependent libraries](https://github.com/baihacker/pe/blob/master/libraries_on_win64.md#build-and-use-pes-dependent-libraries)
  * Build test target:
  > -o a.exe --std=c++20 -fno-diagnostics-color -O2 -march=native -mtune=native -fopenmp -static -Wl,--stack,268435456 -lbf -lflint -lmpfr -lntl -lgmp -lprimesieve -lprimecount -lzmq -lquadmath
* Run:
  * Open terminal and run `test_int128_openmp.bat` and `test_int128_noopenmp.bat`.
  * The performance is affected if the test is triggered by double clicking the bat files from the file explorer.

# PolyMul benchmark
* Test code: [Poly mul test](https://github.com/baihacker/pe/blob/master/test/poly_mul_test.c)

  * flint bn
  * flint pn
  * ntt32 e
  * gmp bn
* Multi-threading is enabled for flint.
  * Note: the multi-threading is enabled when building flint, so it will teake effect even if openmp is disabled.
* Use -O2 option instead of -O3
